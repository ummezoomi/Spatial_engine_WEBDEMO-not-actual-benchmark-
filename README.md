# Spatial_engine_WEBDEMO-not-actual-benchmark-
The cuda spatial engine U.M.E.R was converted to WEBGPU to provide a demo of its temporal coherence (Due to lack of local gpu available), the purpose was to show the temporal coherence. Still in progress
This repository contains videos, benchmarks, visualizations, and web demonstrations related to the development of **Spatial Engine**, a custom simulation and pathfinding framework focused on large-scale agent navigation, spatial partitioning, and real-time environment interaction.

## Project Goals

Spatial Engine explores:

* Large-scale agent simulation
* Efficient spatial partitioning
* Real-time pathfinding
* Navigation benchmarking
* Crowd and flocking behavior
* Interactive WebGPU visualization
